# Recruiter Demo Workflow

This guide details a 5-minute interactive demonstration designed for recruiters, hackathon judges, and technical reviewers.

## Steps
1. **Launch StoryForge AI** (Launches the PySide6 offline desktop application).
2. **Open Demo Universe** (Navigate to the Showcase Dashboard and load "Fantasy Demo" to bypass typing).
3. **Show Story Memory** (Click the *Memory Tab*, demonstrating offline NLP extraction of Characters and Locations).
4. **Show Character Profiles** (Expand a character to reveal Traits, Goals, and Fears).
5. **Show Relationships** (Demonstrate network linking).
6. **Show Story Analysis** (Click the *Analysis Tab* to view structured critiques like "Missing Motivation").
7. **Show Consistency Detection** (Highlight a flagged timeline contradiction without any LLM inference).
8. **Show Universe Graph** (Switch to the *Universe Graph Tab* to view the visual representation of connected lore).
9. **Ask AI**: Type `"Tell me about Arjun"` in the *Chat Tab*.
10. **Show Context Inspector** (Open the inspector to prove that the system automatically dynamically fetched Arjun's Profile and Relationships without stuffing irrelevant lore).
11. **Show Context Sources** (Verify character counts and exact JSON blocks).
12. **Show AI Response** (Review the Qwen response, proving local, secure RAG capabilities).
