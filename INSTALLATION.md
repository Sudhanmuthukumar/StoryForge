# Installation Guide

## Requirements
- Python 3.10+
- PyQt6
- PyYAML

## Setup
1. Clone the repository to your machine.
2. Install dependencies:
   ```bash
   pip install PyQt6 pyyaml
   ```
3. Set your `PYTHONPATH` to the root of the StoryForge repository:
   **Windows (PowerShell):**
   ```powershell
   $env:PYTHONPATH="c:\StoryForge AI"
   ```
4. Run the Creator Suite UI:
   ```bash
   python modules/creator_suite/ui/creator_suite_app.py
   ```
