# Changelog

## [1.0.0] - StoryForge Release
### Added
- **Game Narrative Toolkit**: Full suite of generators (NPCs, Factions, Quests, Campaigns).
- **NPC Intelligence Module**: Relational memory, relationship, goal, and reaction logic for NPCs.
- **World State Engine**: Deterministic calculation of global events and consequence propagation.
- **Creator Suite**: PyQt6 User Interface orchestrating generation models.
- **Unreal Export Layer**: Flat-schema CSV exporter designed for Unreal Engine DataTables.
- **Knowledge Engine**: Underlying pattern datasets harvested from classic literature.
- **Storyforge Core**: Unified telemetry and EventBus architecture.
